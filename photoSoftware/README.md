# 照片处理软件

一款简单易用的照片处理桌面应用框架，支持在 Mac 系统上离线运行。

## 项目状态

⚠️ **当前版本为框架版本**：已搭建基础框架和打包配置，具体功能待实现。

## 核心特性

- 🏗️ **应用框架**：基于 PyQt5 的桌面应用框架
- 📦 **独立打包**：可打包为独立的 Mac 应用程序，无需安装 Python 环境
- 💻 **离线运行**：完全本地运行，无需网络连接
- 🔧 **易于扩展**：清晰的代码结构，方便添加新功能

## 系统要求

- macOS 10.13 或更高版本
- Python 3.7 或更高版本

## 快速开始

### 开发模式（需要 Python 环境）

1. **安装依赖**
   ```bash
   pip3 install -r requirements.txt
   ```

2. **运行程序**
   ```bash
   python3 main.py
   ```

### 打包为独立应用（推荐）

将应用打包为可在任何 Mac 上运行的独立应用程序：

1. **安装打包工具**
   ```bash
   pip3 install pyinstaller
   ```

2. **运行打包脚本**
   ```bash
   ./build_mac.sh
   ```

3. **获取打包结果**
   - 应用程序包：`dist/PhotoSoftware.app`

详细打包说明请查看 [PACKAGING.md](PACKAGING.md)

### 在目标 Mac 上运行

1. 将打包好的应用复制到目标 Mac
2. 双击运行（首次可能需要右键 -> 打开）
3. 无需安装任何环境，可直接运行

## 打包说明

本项目支持打包为独立的 Mac 应用程序，包含所有依赖，可以在没有安装 Python 的 Mac 上运行。

**打包后的应用特点：**
- 无需安装 Python 环境
- 无需安装任何依赖包
- 完全离线运行
- 文件大小约 100-500MB（包含所有依赖）

详细说明请查看 [PACKAGING.md](PACKAGING.md)
